# 슈퍼로봇대전 J 한글패치 도구

GBA 『슈퍼로봇대전 J』(슈퍼로봇대전 J)의 게임 내 텍스트를 한국어로 교체하는 PC용 파이썬 도구 모음입니다. **시나리오 대사**와 **전투(배틀) 대사**를 각각 패치합니다.

> ⚠️ **법적 고지 — ROM 미포함**
> 이 저장소에는 게임 ROM이 **포함되어 있지 않으며, 절대 커밋하지 마세요.**
> 원본 일본판 ROM과 패치된 ROM은 저작권 보호 대상입니다. 본인이 합법적으로 소유한
> ROM에 도구를 직접 적용해 사용하세요. `.gitignore`가 `*.gba` 등 ROM 파일을 자동 제외합니다.

## 요구 환경

- Python 3.8 이상 (Windows / macOS / Linux)
- **scenario/** 단계: `openpyxl` — 번역 매칭 엑셀(.xlsx) 읽기용
- **battle/** 단계: 추가 라이브러리 불필요 (표준 라이브러리만 사용)

```bash
pip install -r requirements.txt
```

## 패치 순서

이 도구들은 **한글 폰트 + 타이틀 로고**가 적용된 ROM을 기준으로 동작합니다. 권장 순서는 다음과 같습니다.

1. **한글 폰트 / 타이틀 로고 적용** — *별도 단계 (이 저장소에 미포함)*
2. **`scenario/`** — 시나리오 대사 패치 → 32MB 한글 ROM 생성
3. **`battle/`** — 위에서 만든 32MB ROM에 전투 대사 패치

> 전투 패치는 입력으로 "시나리오까지 적용된 32MB ROM"을 요구합니다. 일본어 원본(16MB)이나 폰트만 적용한 ROM에는 적용하지 마세요(전투 블록 위치가 다릅니다).

### scenario/ 빠른 실행

```bash
pip install -r requirements.txt          # 저장소 루트에서 1회
cd scenario
python patch_all.py --xlsx-dir translations
```

### battle/ 빠른 실행

```bash
cd battle
python 패치하기.py input.gba srwj_battle_kr.gba
```

(Windows에서 `python`이 동작하지 않으면 `py`로 시도하세요.)

## 동작 원리 (요약)

한글패치 ROM의 폰트는 일본 한자 글리프(2350자) 자리에 한글(가~힝)을 덮어쓴 상태입니다. 따라서 대사에 특정 한자 코드를 넣으면 화면에는 대응되는 한글이 출력됩니다.

```
한국어 → 대응 한자 → 그 한자를 디코딩하는 게임 사전 코드 → 게임 데이터
```

도구는 위 변환을 거친 뒤, 길어진 대사 블록을 ROM의 빈 공간/확장영역에 재배치하고 아카이브 인덱스를 갱신합니다. 블록 내부 구조(포인터 테이블 + 대사)를 그대로 재구성하므로 게임이 정상적으로 읽습니다. 자세한 메커니즘은 각 폴더의 README를 참고하세요.

## 진행 상태

- 시나리오 대사 패치: 동작 (전 챕터)
- 전투 대사 패치: 동작 (unique 대사 약 10,924개 적용)
- 남은 작업: 전체 플레이 검증, 일부 미지원 글자 수정 (`scenario/폰트미보유_문자_수정목록.txt` 참고)

## 라이선스

도구 코드의 라이선스는 추후 지정 예정입니다. 게임 자체의 저작권은 권리자에게 있으며, 이 저장소는 ROM이나 게임 데이터를 배포하지 않습니다.

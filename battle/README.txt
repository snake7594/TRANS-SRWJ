=====================================================================
 슈퍼로봇대전 J (GBA) — 전투 대사 한글 패치 도구
=====================================================================

이 패키지는 '한글 폰트 + 시나리오 + 타이틀'이 적용된 ROM에
전투(배틀) 대사 한글 번역을 입혀 주는 PC용 도구입니다.

unique 대사 11,209개 중 10,924개 번역 적용
(나머지 약 285개는 게임에 안 쓰이는 개발 잔여/깨진 데이터라 원본 유지)


---------------------------------------------------------------------
[1] 필요 환경
---------------------------------------------------------------------
- Python 3.7 이상 (별도 라이브러리 설치 불필요 — 표준 내장 코덱만 사용)
- 윈도우/맥/리눅스 모두 동작


---------------------------------------------------------------------
[2] 준비물
---------------------------------------------------------------------
- 입력 ROM 1개:
    '한글 폰트 + 시나리오 대사 + 타이틀 로고'가 이미 적용된 32MB(33,554,432바이트) ROM.
    (= 이번 프로젝트에서 쓰던 all.gba / srwj_korean_all.gba 와 동일한 그 ROM)
  ※ 일본어 원본(16MB)이나 폰트만 적용한 ROM에는 쓰지 마세요. 전투 블록 위치가 다릅니다.

  이 ROM을 이 폴더에 넣고 이름을 input.gba 로 바꾸면 가장 간단합니다.


---------------------------------------------------------------------
[3] 패치하는 법
---------------------------------------------------------------------
이 폴더에서 명령창(터미널/PowerShell)을 열고:

   python 패치하기.py

  → input.gba 를 읽어 srwj_battle_kr.gba 를 만듭니다.

파일명을 직접 지정하려면:

   python 패치하기.py 내한글롬.gba 결과물.gba

  → 첫 번째 = 입력 ROM, 두 번째 = 출력 ROM(생략 가능)

윈도우에서 'python'이 안 먹으면 'py 패치하기.py' 로 시도하세요.

정상 동작 시 아래처럼 출력됩니다:
   입력 ROM : input.gba (33,554,432 bytes)
   번역 사전: unique 10924개 적용
   변경 블록: 172/177, 번역 토큰: 17187개
   완료! 저장됨: srwj_battle_kr.gba

생성된 srwj_battle_kr.gba 를 mGBA 등 에뮬레이터나 실기에 올려
전투 돌입 시 대사가 한글로 나오는지 확인하세요.


---------------------------------------------------------------------
[4] 동작 원리 (참고)
---------------------------------------------------------------------
- 인코딩: KS X 1001 한글 ↔ SJIS L1 한자 매핑 (한글 폰트가 한자 자리에 한글을 그림)
- 전투 텍스트 블록(아카이브 인덱스 194~370)을 토큰 단위로 번역해 재구성
- 번역으로 길어진 블록만 ROM 확장영역(0x1184A00~)에 순차 재배치하고
  아카이브 인덱스만 갱신 (ARM 코드 패치 불필요)
- 번역 안 된 토큰/비대사는 원본 바이트 그대로 → 원본과 왕복 항등
- 줄바꿈(\n)·전각공백·이름변수 코드([7d]~[80]) 위치를 원문과 동일하게 보존
  (토큰 수가 바뀌면 프롤로그/이벤트가 멈출 수 있어 구조를 그대로 유지)


---------------------------------------------------------------------
[5] 번역을 직접 고치고 싶다면
---------------------------------------------------------------------
battle_dialogue_unique.json 을 편집하세요. 형식:

   [ { "jp": "행くぜ！", "tr": "간다!" }, ... ]

- "tr" 값만 바꾸면 됩니다. "jp"는 건드리지 마세요(매칭 키).
- 줄바꿈은 \n, 전각공백은 \u3000 으로, [7d]/[7e]/[7f]/[80] 코드는 위치 그대로 두세요.
- "tr"을 빈 문자열("")로 두면 그 대사는 번역하지 않고 원본을 유지합니다.
- 수정 후 다시 python 패치하기.py 만 실행하면 반영됩니다.


---------------------------------------------------------------------
[6] 포함 파일
---------------------------------------------------------------------
  패치하기.py                  ← 실행 런처 (이것만 돌리면 됨)
  srwj_battle_kr_insert.py     ← 한글 삽입 엔진 (메인)
  srwj_battle_codec.py         ← 전투 텍스트 코덱
  srwj_archive.py              ← 아카이브 인덱스/재배치 기본기
  kouji_tr.py                  ← 마징가계 기체/기술명 폴백 번역
  battle_dialogue_unique.json  ← 번역 데이터 (10,924개)
  README.txt                   ← 이 파일

※ 입력 ROM과 결과 ROM(srwj_battle_kr.gba)은 용량/저작권상 이 zip에 넣지 않았습니다.
  (결과 ROM은 별도로 이미 전달되었습니다.)
  반드시 작업 전 원본 ROM을 백업해 두세요.
